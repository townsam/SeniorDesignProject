# SeniorDesign (Unity + ML-Agents)

This repository contains the Unity project files needed to review current progress (scenes, scripts, prefabs, configs, and docs).

## What to open
- Unity Editor version: **6000.2.7f2** (see `ProjectSettings/ProjectVersion.txt`)
- Start scene: `Assets/Scenes/MainMenu.unity`

## Key docs
- Demo / run steps: `Docs/DEMO_CHECKLIST.md`
- Scene wiring checklist: `Docs/Scene_Inspector_Checklist.md`
- Playable roadmap: `Docs/Playable_State_Roadmap.md`
- Engineering handover log: `README_HANDOVER.md`
- Mac setup (Unity + optional trainer): `README_MAC_SETUP.md`

## What is intentionally not committed
Generated and machine-specific folders are excluded (Unity `Library/`, `Temp/`, `Logs/`, `UserSettings/`), local Python environments (`.venv-mlagents/`), and generated ML training outputs (`results/`).
